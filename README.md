# 🎓 CampusLink - Placement Interaction System

CampusLink is a full-stack web application designed to streamline the recruitment process within universities. It allows students to register, view job openings, and track their applications in real-time.

---

## 🚀 Features
* **Student Dashboard:** View and apply for upcoming placement drives.
* **Real-time Registration:** Secure student profile creation with CGPA and skill tracking.
* **Admin/Recruiter API:** Endpoints to post new job roles and packages.
* **Responsive UI:** Built with React and Tailwind CSS for a seamless experience.

---

## 🛠️ Tech Stack
**Frontend:** React.js, Tailwind CSS  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**Version Control:** Git & GitHub
